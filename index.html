Projects

BlocksHub (June 2019 - November 2020)

BlocksHub (formally "Econtest" and "Froastmons") was a small project between me and another partner. The website was essentially trying to be a clone of Roblox: we had groups, forums, games (albeit primitive), two in-game currencies, clothing, hats, and more. The partner eventually left, and after about a year of inactivity, the site was shut down. It has been open sourced, and is available on GitHub.

The project mostly used the following tools:

  • NodeJS/Javascript
  • Typescript
  • TsED/Express
  • MySQL 5.7
  • Redis
  • Bootstrap 4

Some screenshots of a BlocksHub game and the BlocksHub dashboard

RblxTrade (January 2019 - Present)

RblxTrade is a Roblox trading fan site. Some notable features, not present on other sites, include the ability to view terminated player's inventories, terminated player profiles, and more. The site was originally created by me in January 2019, however, I later sold the site in the summer of 2020. Later in Fall/Winter 2020, I was invited back to be a developer (alongside another developer), which is what my main focus has mostly been since then.

Things I've created at .trade include:

  • The website backend
  • The scanning processes
  • The discord bots

Economy Simulator (January 2021 - Present)

economy-simulator is a simulation website for a popular video game. To prevent abuse, you will have to join our Discord Server to learn about it.
